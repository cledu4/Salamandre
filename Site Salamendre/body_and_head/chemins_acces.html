Les chemins d'accès : la fiche


1) Le chemin absolu

Le chemin absolu, c'est l'endroit exact dans un ordinateur, un disque dur, une clef USB, où se trouve un document.

Exemple : E:\Technologie\site\page1.html

Ainsi, il n'existe qu'un seul et unique lien absolu pour un document. Pour le trouver, le plus simple est de faire CLIC GAUCHE --> Propriétés.

2) Le chemin relatif

Le chemin relatif, c'est l'endroit où se trouve un document par rapport à un autre document.

Si le document cherché est dans le même dossier, il suffit de taper le nom du fichier.
Si le document cherché est dans un autre dossier :
  • Dans un dossier au-dessus du document : ../nom_fichier
  • Dans un dossier en dessous du document : ./nom_dossier/nom_fichier


Ces méthodes peuvent être combinées : pour sortir du dossier, puis entrer dans un autre afin de trouver le document, on peut écire ../nom_dossier/nom_fichier
