Bienvenue sur la Salamandre Informatisée !

Toutes les pages HTML sont construites sous la forme <head>, puis <body>. Tout d'abord, intéressons-nous à la partie <head>.

1) Head


La partie Head a de nombreuses fonctions. Afin d'être pratique et simple, ce site donne les informations nécessaires à la création d'un site.
Voici les balises présentes dans la partie <head> de ce site :
  • <meta charset="UFT-8"/>
  • <title>La Salamandre Informatisée</title>
  • <link rel="shortcut icon" href="../images/salamandre.gif"/>
  • <link href="../css/test.css" rel="stylesheet"/>

Ligne par ligne, la partie Head est détaillée si dessous :
  • Ligne 1 : encodage des caractères
  • Ligne 2 : titre de la page (dans l'onglet)
  • Ligne 3 : icone de la page (dans l'onglet),
  • Ligne 4 : lien avec la feuille de style CSS (voir partie CSS)
Attention ! La partie <head> d'une page HTML peut être vide, même si cela n'a pas un grand intérêt.

Pour créer un site complet, il est recomandé de recopier la ligne 1 exactement : <meta charset="UFT-8"/>.

La ligne 2 s'écrit ainsi : <title>Titre dans l'onglet</title>

L'utilité de la ligne 3 est discutable : elle améliore le rendu du site en affichant un dessin dans l'onglet.
Elle s'énonce ainsi : <link rel="shortcut icon" href = "dossier/image.gif"/>
Vous pouvez utiliser les 2 types d'accès: les chemins d'accès relatifs ou les chemins d'accès absolus.
