Les fonctions


Il est possible d'intégrer dans une page HTML des programmes afin de ajouter des fonctionnalités. Une fonction est toujours écrite dans la partie <body>. Voici un exemple possible de fonction.

...
<span onmousedown="fonction()">test</span>
<script>
function fonction()
{
alert("hello_word")
return;
}
</script>

Voici le résultat :test

Après l'éxecution de la fonction, c'est-à-dire après avoir cliquer sur le mot "test" (onmousedown = cliquer), la page renvoie "hello_word".
Tout cette partie rejoint la page "lien et boutons" partie "alerte".
Il existe d'autre type de fonction et d'autres utilisations, mais elles sont pas abordées dans ce site.
