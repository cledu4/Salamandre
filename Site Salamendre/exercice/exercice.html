Le mot "HTML" signifie :
- Hyper Text Markup Language.
- Hyper Test Margin Language.
- Hyper Text Margin Label.

La balise </br> ou <br> sert à :
- taper du texte.
- sauter une ligne.
- à donner un attribut.

Vrai ou faux ? Une page HTML doit obligatoirement commercer par <! DOCTYPE html>.
- Vrai
- Faux

Vrai ou faux ? Est-ce que une balise <span> peut-elle être vide (sans attribut) ?
- Vrai
- Faux

Une page CSS reliée à une page HTML part <link href="test.css" rel="stylesheet"/> sert à :
- Intégrer plus de texte.
- Créer des symboles inexistants dans une page normal (ex: Σ ).
- Mettre en forme le texte.

Une fonction dans une page HTML doit toujours commencer par :
- <span ... />
- <script> ... </script>
- <div> ... </div>

A quoi sert l'attribut "class" dans une balise <span ... /> ?
- à mettre en forme (avec le CSS).
- à rendre le texte rouge.
- à mettre le texte en italique.

Vrai ou Faux ? La balise <head> peut-elle être vide ?
- Vrai.
- Faux.

Quel est la bonne écriture ? <span class:"souligner"> Faux </span>
- <span clas:"souligner"> Faux </span>
- <span class="souligner"> Faux </span>
- <span class="souligner"> Faux <span>

Quel est le rendu de <u> test </u> ?
- Test
- Test
- Test

A quoi sert la balise : <meta charset="UFT-8"/> ?
- changer la couleur de l'écriture.
- mettre un lien.
- à l'encodage des caractères.
