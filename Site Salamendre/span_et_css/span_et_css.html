Cette page comporte 2 thèmes différents mais liés : la balise <span> et le css.

1) La balise <span>


La balise <span> est une balise "neutre", c'est à dire qu'écrire <span>test</span> ne change rien au mot "test".

L'avantage d'utiliser cette balise est la possibilité de lui donner des attributs : dans la partie "Body Head" du site, la balise <span> est utilisée afin de rendre un texte coloré.

Voici quelques utilisations possibles de la balise <span> :
  • Texte en couleur :<span style="color:couleur"> (la couleur doit être en anglais)
  • Texte en couleur :<span style="color:#couleur"> (la couleur doit être en hexadécimal)

Les balises <span> ont une autre fonction : celle de pouvoir associer à une "classe" une suite d'atributs.

Dans la feuille de style (informations dans la seconde partie), vous pouvez créer une classe, c'est-à-dire que tous les balises <span> qui auront en attribut la classe auront les attributs que cette classe a dans le CSS.

Dans le <span> :
<span class="la_classe">...</span>

Dans le CSS :
.la_classe {
text-decoration:underline;
}
