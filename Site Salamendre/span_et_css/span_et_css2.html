2) Le CSS


Le CSS permet de mettre en forme le texte, comme vu dans la page précédente. Ici, nou allons voir comment l'utiliser.

Pour intégrer la feuille de style (c'est la feuille de CSS), il suffit de mettre, dans la partie <head> :
<link href="nom_fichier.css" rel="stylesheet"/>

Pour mettre des attributs dans le body d'une page, il faut écrire :
body {
...
}

Voici une autre liste des atributs qui peuvent vous servir :
  1. background-color (arrière-plan coloré) peut avoir comme valeur "red", "blue", "gray", ... dans le .body
  2. padding (marge) comprend 3 valeurs : marge du haut, marge des côtés, marge du bas dans le .body
  3. background-repeat (répétition de l'image de fond) est soit "repeat", "repeat-y", "repeat-x" ou "no-repeat" dans le .body
  4. background-image (image de fond) est intégrer par un lien (relatif ou absolu) dans le .body
  5. font-size (taille de police) requiert une valeur suivi de "px" (pixels) dans le .span
  6. text-decoration (soulignement) a de nombreuses valeurs, comme "underline" (souligner), "overline"(trait au-dessus) dans le .span


Un attribut est toujours écrit sous la forme :
attribut:valeur;
